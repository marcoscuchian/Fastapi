# Pasos a seguir para levantar el proyecto

- Clonar repositorio.

- Crear entorno virtual:

``` python3 -m venv ./venv ```

- Activar

``` . venv/bin/activate ```

- Instalar requirements

``` pip install -r requirements.txt ```
